Minesweeper - Starter

This is a simple Minesweeper web starter you can open in any modern browser.

How to run:
1. Open `index.html` in your browser (double-click or use a local server).
2. Choose a difficulty preset (Beginner / Intermediate / Expert) or click the smiley to start a new game.
3. Left-click to reveal a cell; right-click to flag (on touch devices use long-press or emulate right-click).

Status / next steps:
- Difficulty presets: Completed (Beginner/Intermediate/Expert are available).
- Timer, mine counter, best scores: Not implemented yet.
- Unit tests for `app.js`: Not implemented yet.
